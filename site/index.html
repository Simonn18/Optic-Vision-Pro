Le flux de travail en 4 étapes

Optic Vision Pro guide l'analyse de bout en bout. Une barre de statut rappelle l'étape courante en bas de la fenêtre.

Étape 1

Chargement des dossiers

Au lancement, l'écran d'accueil propose deux modes — PRÉ-TRAITEMENT et TRAITEMENT D'IMAGE — ainsi qu'une bascule de langue English / Français.

L'analyse démarre par un dossier de travail. Depuis la barre du haut, deux boutons : Ouvrir un dossier de travail et Ouvrir un dossier d'images. Tant qu'aucun dossier n'est chargé, la zone centrale invite à « Ouvrez un dossier pour afficher les images ».

Structure d'un dossier de travail

Un dossier de travail valide regroupe les sous-dossiers attendus par l'application :

  • fundus_images — les photographies du fond d'œil à analyser.
  • segmentation_masks — les masques (veines, artères, disque optique).
  • fundus_rendu_images_finales — les rendus visuels générés.
  • results — les fichiers de mesures exportés.
Étape 2

Segmentation & lancement des mesures

Une fois les images chargées, le panneau Segmentation (à gauche) pilote l'affichage des couches et l'édition du disque optique. La barre de miniatures en bas permet de naviguer (◀ Précédent / Suivant ▶).

Tutoriel : vérifier puis lancer les mesures

  1. 1

    Vérifiez les miniatures

    Avant tout calcul, parcourez la barre de miniatures en bas de la fenêtre et contrôlez, pour chaque image, que le disque optique (repère vert) est bien placé — centré sur la papille, au bon diamètre. C'est lui qui définit les zones concentriques : seules les images dont le disque est correctement positionné seront mesurées.

    💡 Si un disque est décalé, sélectionnez sa miniature, corrigez-le avec Éditer le disque, puis Valider les disques.

  2. 2

    Validez les disques, puis lancez les mesures

    Appuyez sur Valider les disques, puis sur Lancer les mesures. Le calcul se déclenche pour le lot et le panneau de mesures s'ouvre.

    ⏱️ Lancez les mesures dès le début de votre session : le calcul peut être très long et se poursuit pendant que vous travaillez.

  3. 3

    Laissez tourner en arrière-plan

    Les mesures s'exécutent en arrière-plan : l'application reste utilisable pendant le traitement et la barre de statut affiche l'avancement (« Calcul global en cours, veuillez patienter… »). À la fin, un message confirme « Mesures terminées » et les résultats apparaissent dans le panneau.

Édition du disque

Créer un disque, Éditer le disque puis Valider les disques — il sert de référence aux zones.

Couleurs

Personnaliser la couleur des veines, artères, du disque et des superpositions.

Opacité

Régler l'opacité de chaque couche individuellement selon sa visibilité.

Réinitialiser

Le bouton Réinitialiser les paramètres remet couleurs et opacités d'origine.

Les couches Veines, Artères et Disque optique sont cochées par défaut ; décochez celles dont vous n'avez pas besoin. La couleur de chaque segmentation — et celle de la superposition artères/veines — est personnalisable.

Étape 3

Mesures

Le panneau Mesures (à droite) configure le calcul. On choisit le type de vaisseaux, les zones concernées et les groupes de métriques, puis on exporte ou on affiche les résultats.

🩸

Type de vaisseaux

Veines, Artères, ou Artères + Veines.

⭕

Zones

Zone A, B, C, All ou Out — anneaux concentriques autour du disque.

📐

Groupes de mesures

Contrôle qualité, Détection vaisseaux, Métriques géométriques.

⚠️ Les paramètres sont organisés en arborescence : vous devez sélectionner au moins un paramètre par bloc (type de vaisseaux, zones, groupes de mesures) pour pouvoir afficher ou exporter.

Afficher & exporter les résultats

👁️

Afficher les mesures

Afficher les mesures montre les valeurs de l'image active pour les paramètres choisis. Pour voir une autre image : cliquez sur sa miniature puis ré-appuyez sur Afficher les mesures.

📤

Exporter les mesures

Export au format texte — pour toutes les images ou seulement l'image active à l'écran, selon les paramètres sélectionnés.

Zones concentriques autour du disque optique

ZoneRayonDescription
ODRDisque optique — la référence.
Zone A2 × RPremier anneau péripapillaire.
Zone B3 × RAnneau intermédiaire.
Zone C5 × RAnneau le plus large.

Pour chaque zone et chaque quadrant (Nord, Est, Sud, Ouest), l'application calcule l'aire, l'aire visible et des métriques de vaisseaux (nombre de croisements, calibre moyen…). Exemple de sortie : nbXFC_B, meanCaliberF, sdBigXCaliber…

Étape 4

Sauvegarde & export

La sauvegarde se fait dans le dossier choisi via Ouvrir un dossier de travail. Les résultats y sont organisés en 4 sous-dossiers.

Structure de sauvegarde

  • fundus_images — les images d'origine, avec un JSON de paramétrage (couleurs et opacités).
  • fundus_rendu_images_finales — les images PNG : l'image d'origine avec les segmentations paramétrées superposées.
  • results — un fichier de mesures globales + un fichier JSON par image.
  • segmentation_masks — les masques de segmentation (veines, artères, disque optique).

Les options de sauvegarde

💾

Sauvegarder

Enregistre l'image en cours de traitement selon la structure ci-dessus.

🗂️

Sauvegarder toutes les images

Enregistre tout le dossier, au choix : un dossier par image (5 images → 5 dossiers) ou un seul dossier pour toutes les images.

✅

Sélection

Sélection crée un dossier à côté avec un sous-ensemble d'images : cliquez directement sur les miniatures, un cadre vert entoure les images choisies.

Bon à savoir

Optic Vision Pro travaille directement dans le dossier d'images pour vous éviter de tout relancer.

🔄

Paramètres synchronisés

Les modifications se font dans le dossier d'images : les paramètres y sont mis à jour à chaque fois.

🖼️

Mise à jour des rendus

Pour rafraîchir fundus_rendu_images_finales, cliquez sur Sauvegarder les rendus.

⚡

Mesures conservées

Si des mesures existent déjà dans le dossier, le panneau Mesures s'affiche directement — pas besoin de relancer le calcul.

Installation & lancement

L'application repose sur Python 3. Pour l'exécuter depuis les sources :

# 1. Installer les dépendances
pip install -r requirements.txt

# 2. Lancer l'application
python newapp.py

Pour générer un exécutable autonome :

# Avec PyInstaller
pip install pyinstaller
pyinstaller newapp.spec

# ou avec Nuitka
./build_nuitka.sh
